# ============================================
# 16. COMMON DEPENDENCY ISSUES & SOLUTIONS
# ============================================

## Issue: Flutter not found
Solution: Add Flutter to system PATH
- Windows: Environment Variables → PATH → Add flutter\bin
- Mac/Linux: Add to ~/.bashrc or ~/.zshrc

## Issue: Android licenses not accepted
Solution: flutter doctor --android-licenses
Accept all licenses

## Issue: PHP PDO not enabled
Solution: Edit php.ini
Uncomment: extension=pdo_mysql

## Issue: MySQL won't start
Solution: Check port 3306 not in use
Change port in XAMPP config if needed

## Issue: Gradle build fails
Solution: 
- Check internet connection
- Clear gradle cache
- Update Android SDK
